## 2024-05-18 - Accessibility: Added aria-labels to cart buttons\n**Learning:** The cart page uses icon-only buttons for quantity control (+ / -) and item removal (x). These lacked aria-labels, making it difficult for screen reader users to interact with the cart effectively.\n**Action:** Ensure that all icon-only buttons, especially in critical flows like the shopping cart, have descriptive `aria-label` attributes to improve accessibility.
